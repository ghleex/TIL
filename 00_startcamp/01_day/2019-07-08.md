## 4. Python 심화

### CLI란? Command Line Interface

- 반대말:  GUI(Graphic User Interface)

- CLI 종류: Unix Shell(sh, zsh, bash 등), CP/M, DOS의 command.com 등

##### BASH 명령어

- ls: 현재 디렉토리 내용들 나열
- cd: 현재 작업하는 디렉토리 변경
- mkdir: 새로운 디렉토리 생성
- rm: 파일 지우기
- exit: 터미널 종료
